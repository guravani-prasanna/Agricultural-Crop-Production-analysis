## 📖 Abstract

This project analyzes agricultural crop production across different states and districts of India using data analytics techniques. The objective is to identify production trends, compare crop yields, analyze seasonal patterns, and present insights through SQL analysis and Tableau dashboard planning. The project demonstrates the complete data analytics workflow, including data collection, preparation, analysis, visualization, and documentation.

## 📌 Project Overview

Agriculture is one of the most significant sectors of the Indian economy. This project analyzes agricultural crop production across different states and districts of India using SQL and Tableau. The objective is to identify production trends, high-yield regions, major crops, and seasonal patterns through interactive dashboards and visualizations.

This project was developed as part of the **Skill Wallet Data Analytics Program**.

---

# 🎯 Objectives

- Analyze agricultural crop production in India.
- Identify the highest and lowest producing states.
- Compare crop production across seasons.
- Study cultivated area and production trends.
- Build interactive Tableau dashboards.
- Present actionable insights using data visualization.

# 📌 Key Insights

- Rice and Wheat are the highest produced crops.
- Uttar Pradesh contributes significantly to crop production.
- Kharif season records the highest production.
- Crop production has increased over the years.
- Some states have high cultivated area but comparatively lower productivity.
