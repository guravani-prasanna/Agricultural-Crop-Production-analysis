# Dashboard Description

## Project Title
India's Agricultural Crop Production Analysis

## Dashboard Objective
The dashboard provides an interactive visualization of agricultural crop production across different states and districts of India.

## Dashboard Components

### 1. KPI Cards
- Total Production
- Total Cultivated Area
- Total States
- Total Crops

### 2. Charts
- State-wise Crop Production (Bar Chart)
- Crop-wise Production (Pie Chart)
- Year-wise Production Trend (Line Chart)
- Season-wise Production (Bar Chart)

### 3. Filters
- State
- Crop
- Season
- Crop Year

## Expected Outcome

The dashboard helps users identify:
- Top producing states
- Major crops
- Seasonal trends
- Production growth over the years
- Overall agricultural performance
